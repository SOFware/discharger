# Discharger
Code supporting tasks that discharge code for deployment.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "discharger"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install discharger
```
